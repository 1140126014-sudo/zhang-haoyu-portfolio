- ARIA 交互检查不能只看属性是否存在，还要看属性指向的目标在折叠态是否仍然存在。常驻 DOM + `hidden` 更适合可反复展开的作品集面板。

### 可复用经验

- 动效能力可以保留，但不应默认压进主入口包；对 GSAP、ScrollTrigger 这类重动效依赖，优先使用组件级 dynamic import。
- reduced-motion 不是只在 effect 里关动画，还应尽量在首轮渲染前决定是否挂载重动效入口。
- 自动化发现的重复问题应沉淀成脚本，而不是每次质检手工重写扫描命令；`npm run check` 是后续复盘的最低稳定入口。
- 资源扫描必须贴合项目真实写法，例如本项目同时存在 `asset()`、`/assets/...` 字符串、CSS `url()` 和 `OPTIMIZED_IMAGE_SOURCES` 推导规则。
